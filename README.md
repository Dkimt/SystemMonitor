# SystemMonitor
分布式集群监控系统
> python开发的一个基于分布式的集群监控系统，C/S架构

主要监控的系统属性有：
   - 内存使用率
   - CPU占用率
   - 磁盘I/O
   - 磁盘使用率
   - 外网延迟
   - 主机启动的端口信息
